## Quick Start

### Checkout pongboard:
    https://github.com/drewrothstein/pongboard.git

### Get Meteor and Meteorite
Install [meteor.js] (https://www.meteor.com/) and install [meteorite] (http://oortcloud.github.io/meteorite/).

### Run
    cd pongboard/app
    mrt
